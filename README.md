# Ascent

**Ascent** is a personal growth web application designed to help users track habits, set and achieve goals, reflect through journaling, and gain insights into their progress. It combines motivational features with visual progress tracking to encourage consistency and personal development.

## Features

- **Daily Habits & Goals**  
  Track your daily habits, mark them as complete, and monitor your streaks over time. Goals can be categorized for better organization.

- **Journal & Mood Tracking**  
  Reflect on your day with daily prompts and select moods to track emotional well-being. Save entries to review your personal growth.

- **Insights & Analytics**  
  View progress through visual components like progress rings and statistics cards to stay motivated.

- **Motivational Quotes**  
  Receive daily quotes for inspiration and reflection.

- **Responsive & Interactive UI**  
  Clean and modern interface with collapsible navigation, gradient designs, and subtle animations.

## Technologies Used

- React 19
- React Router DOM 7
- Vite
- JavaScript (ES6+)
- HTML5 & CSS3 (with Flexbox and CSS Grid)
- LocalStorage for state persistence
- External API for motivational quotes
